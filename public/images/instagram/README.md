# Instagram Images

Please place the following Instagram post images in this directory:

- `vanilla-gelato.jpg` - Vanilla gelato post
- `strawberry-gelato.jpg` - Strawberry gelato post  
- `mango-gelato.jpg` - Mango gelato post
- `pistachio-gelato.jpg` - Pistachio gelato post
- `gelato-making.jpg` - Behind the scenes video thumbnail
- `shop-interior.jpg` - Shop interior post

## Image Requirements

- **Format**: JPG or WebP
- **Size**: 400x400px (square format for Instagram grid)
- **Optimization**: Compress for web (aim for < 100KB per image)
- **Quality**: High resolution, well-lit photos

The website will automatically use these images in the Instagram feed section.
