# Desserts Images

Please place the following dessert images in this directory:

- `lemon-tart.jpg` - Lemon tart dessert
- `cheesecake.jpg` - Cheesecake dessert

## Image Requirements

- **Format**: JPG or WebP
- **Size**: 400x400px (square format)
- **Optimization**: Compress for web (aim for < 100KB per image)
- **Quality**: High resolution, well-lit photos

The website will automatically use these images in the desserts section.
