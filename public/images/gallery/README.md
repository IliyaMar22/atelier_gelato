# Gallery Images

Please place the following images in this directory:

- `gallery1.jpg` - Gelato display
- `gallery2.jpg` - Shop interior  
- `gallery3.jpg` - Gelato making process
- `gallery4.jpg` - Tiramisu special
- `gallery5.jpg` - Customer favorites

## Image Requirements

- **Format**: JPG or WebP
- **Size**: 800x600px minimum
- **Optimization**: Compress for web (aim for < 200KB per image)
- **Quality**: High resolution, well-lit photos

The website will automatically use these images in the gallery section.
