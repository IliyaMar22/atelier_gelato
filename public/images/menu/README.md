# Menu Images

Please place the following menu images in this directory:

- `gelato-menu.jpg` - Gelato flavors menu
- `desserts-menu.jpg` - Desserts menu
- `beverages-menu.jpg` - Beverages menu

## Image Requirements

- **Format**: JPG or WebP
- **Size**: 800x600px minimum
- **Optimization**: Compress for web (aim for < 200KB per image)
- **Quality**: High resolution, clear text, well-lit photos

The website will automatically use these images in the menu section.
